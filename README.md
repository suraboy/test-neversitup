# test-neversitup
Testing in Back-End Developer (Golang) Position

Getting Started
-----------
#### Requirements

To run this application on your machine, you need at least:

- go 1.20


Sync lib
---------------------

```
make tidy
```

Running Application
------------------------------------
Start Project

```bash
make run
```

Testing Application
------------------------------------
Test
```bash
make test
```
Test Coverage
```bash
make test-coverage
```
